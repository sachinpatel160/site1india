== Description ==

The Politics WordPress theme is designed for politicians, political causes, and government agencies to clearly represent themselves online with a number of useful features. Those features include multiple home page sections that can be easily moved and edited.

You can customize the theme with the native WordPress Customize feature allowing changes to appear instantly while editing. Full support for contact form, post sharing, and gallery displays with the Jetpack plugin.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Home Page ==

1. To use the home page template, create a new page called "Home" and assign it the "Front Page" template under the Page Attributes settings.
2. Navigate to "Settings > Reading" and assign Front page as "Home" and Posts page as "Blog".

== Home Hero Section ==

1. After assigning the home page template as described above, set the top hero image from "Appearance > Customize > Home"
2. Add overlaying content by placing a widget in the "Home Hero" widget area from "Appearance > Widgets".
3. To add an email sign-up form, activate the [Mailbag](https://wordpress.org/plugins/mailbag/) plugin and paste its shortcode into a text widget.

== Blog Page ==

1. Create a new page and publish it.
2. Navigate to "Settings > Reading" and assign Posts page as the blog page you just created.
